# php-js-exercices

Exercício de UPDATE - Será criado 4 DIVs cada um com um nome pré-definido, DIV1="Rio Cautário", DIV2="Rio Pacáas Novos", DIV3="Tambaqui" e DIV4="Rio Madeira" (Essas DIVs devem ter obrigatoriamente esses nomes de inicio). Após a estilização e criação do Front-End, o Back-End terá que fazer um formulário que faz um UPDATE no Banco de Dados em uma coluna chamada "predios", e depois do UPDATE retornar a resposta para o HTML.
